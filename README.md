# Sim Max

Sim Max 是一款完全免费的 SIM / eSIM 号码资产管理、会员订阅管理与保号提醒 Android 工具。

---

## 📱 主要功能

### 号码管理

- **SIM / eSIM 号码记录**：添加、编辑、删除号码，记录运营商、国家区号、手机号码、套餐余额、备注等
- **国家区号库**：覆盖 167 个国家/地区，支持 163 张真实国旗图片
- **运营商自动识别**：输入号码后自动推荐运营商，支持手动选择
- **到期日期管理**：设置到期日期、保号周期（7/15/30/90/180/365天），支持长期号码标记
- **eSIM 激活信息**：记录 EID、SM-DP+ 地址、激活码，支持二维码展示和相册扫描
- **号码搜索与筛选**：按运营商、国家、号码搜索，支持到期状态筛选（正常/即将到期/已过期）
- **排序功能**：按到期时间远到近 / 近到远排序
- **信号状态**：在线/离线/漫游/无服务状态标记
- **卡片国旗背景**：号码卡片右侧展示对应国家的丝绸风格国旗背景

### 会员订阅管理

- **会员记录**：添加和管理各类订阅服务（流媒体、软件服务、云存储、VPN、音乐、教育、游戏、新闻资讯等）
- **分类筛选**：9 大分类标签等宽网格排列，点击快速筛选
- **订阅类型**：月付/季付/年付/终身/自定义
- **续费管理**：设置续费金额、续费周期（天）、提前提醒天数
- **自动续费/到期提醒**：独立开关控制
- **线性图标**：10 种线性图标（手机/消息/音乐/电视/云/游戏手柄/包裹/扳手/笔记本/学士帽）可选
- **毛玻璃卡片**：会员卡片采用与号码卡片一致的毛玻璃渐变效果

### 到期提醒

- **本地通知**：Android 本地推送通知，支持自定义提前提醒天数和提醒时间
- **Telegram 通知**：配置 Bot Token 和 Chat ID，到期自动推送 TG 消息
- **SMTP 邮件**：配置邮箱服务器，到期自动发送邮件提醒
- **云端提醒服务**：独立 API Key 隔离数据，服务器每 30 分钟检查一次到期，即使 App 被杀后台也能发送提醒
- **自动同步**：新增/编辑/删除/保号/导入和配置变更后自动同步到云端

### 刷流量测试

- **真实下载测试**：选择任意号码执行真实的下载流量测试，验证号码数据套餐是否正常
- **自定义接口**：支持自定义流量测试 URL 和下载大小
- **Telegram 推送**：测试结果可自动推送到 Telegram

### 数据管理

- **JSON 导出**：生成完整 JSON 备份文本，支持复制和分享
- **CSV 导出**：生成 CSV 表格文本
- **JSON / CSV 导入**：粘贴 JSON 或 CSV 数据恢复号码列表
- **云端同步**：一键同步全部数据到云端服务器

### 外观与个性化

- **深色模式**：全面适配深色主题，所有页面、卡片、弹窗、输入框、标签均支持深浅色切换
- **毛玻璃效果**：号码卡片和会员卡片均采用毛玻璃渐变叠加效果
- **自定义背景**：支持设置自定义背景图片，可调节遮罩透明度
- **国旗显示**：首页卡片可选择是否显示国旗背景

### 多语言支持

- 简体中文（默认）
- 繁体中文
- English
- 日本語
- 阿拉伯语（支持 RTL 右到左布局）

## 🏗 技术架构

- **语言**：Kotlin
- **UI 框架**：Jetpack Compose + Material3
- **数据存储**：SharedPreferences（本地 JSON）
- **架构**：单 Activity + Composable 组件
- **图标**：全部使用 Canvas 线性绘制，无外部图标库依赖
- **最低 SDK**：Android 8.0（API 26）
- **目标 SDK**：Android 14（API 34）

## 🔔 云端提醒服务

App 默认使用云端地址：

```text
https://ccs.ziranaa.top:16670
```

服务端源码在：

```text
server/simjiang-reminder/
```

### 工作原理

1. 手机端生成唯一 API Key
2. 开启自动同步后，号码数据变更自动推送到云端
3. 服务端每 30 分钟检查一次到期号码
4. 到期时通过 Telegram / 邮件发送提醒
5. 即使手机 App 被杀后台，云端仍可正常提醒
